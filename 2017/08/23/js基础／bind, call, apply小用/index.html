发表于 2017-08-23   |   分类于 js   |  

如何使 Cat.sing 输出 wang ？

使用bind, call, apply实现很简单（需注意bind与call/apply的区别，call/apply会立即执行，而bind不会）

  1. Cat.sing.bind(Dog)()
  2. Cat.sing.call(Dog) / Cat.sing.apply(Dog)

如果不能使用bind，那么我们如何实现一个函数mybind使得 Cat.sing.mybind(Dog)() 输出 wang ？可以使用call/apply实现
